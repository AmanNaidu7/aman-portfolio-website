# Portfolio Website

Production Next.js portfolio website for Aman Naidu.

## Structure

- `src/` application code
- `content/` markdown-managed site content
- `public/` runtime static assets

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This repository is ready for Vercel with the default Next.js settings:

- Framework Preset: `Next.js`
- Root Directory: `.`
- Build Command: `npm run build`
- Install Command: `npm install`
